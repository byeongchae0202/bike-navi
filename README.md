# 모바일 웹앱 프론트엔드 스캐폴딩

React + TypeScript + Vite 기반으로 모바일 우선 UI 스캐폴딩이 구성되어 있습니다.

## 실행

```bash
npm install
npm run dev
```

## 포함된 기본 구조

- **홈**: 현재 위치 지도 표시(OpenStreetMap 임베드) + 위치 기반 추천 코스
- **탐색**: 거리/오르막 필터 검색, 지도 클릭으로 목적지/경유지 선택, 경로 안내 시작
- **일정**: 실제 안내 종료 시 저장되는 주행 거리/시간/평균속도 기록
- **마이페이지**: 키/몸무게/나이 프로필 입력, 실제 기록 기준 주/월/년 거리 통계 보기
- 하단 탭 네비게이션(홈/탐색/일정/마이)
- 모바일 메타 뷰포트 설정(`index.html`)

## 동작 방식

- 위치/속도는 브라우저 Geolocation API를 사용합니다.
- 경로 안내는 OSRM 공개 라우팅 API를 사용해 계산합니다.
- 실제 주행 기록/프로필은 브라우저 로컬 스토리지에 저장됩니다.

## 빌드

```bash
npm run build
```
